Server Setup

Configure the JC Budgeting Server, confirm the address other devices should use, and watch the live server log output in one place.

Server databases are stored in the server Databases folder. Uploaded databases keep their source file name when possible, and duplicates get a timestamp suffix automatically.
On Umbrel or another shared server, the Primary Access Address is the address your JC Budgeting client should use instead of localhost.
